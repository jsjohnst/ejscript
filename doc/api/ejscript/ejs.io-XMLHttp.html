public open(method: String, url: String, async: Boolean, user: String, password: String): Void

Open a connection to the web server using the supplied URL and method.

Parameters
method: String HTTP method to use. Valid methods include "GET", "POST", "PUT", "DELETE", "OPTIONS" and "TRACE"
url: String URL to invoke
async: Boolean If true, don't block after issuing the requeset. By defining an $onreadystatuschange callback function, the request progress can be monitored. [default: false]
user: String Optional user name if authentication is required.
password: String Optional password if authentication is required.
get public readyState(): Number

Return the readystate value.

Description
This value can be compared with the XMLHttp constants: Uninitialized, Open, Sent, Receiving or Loaded.
Returns
Returns a number with the value: Uninitialized = 0, Open = 1, Sent = 2, Receiving = 3, Loaded = 4.
