README.TXT for the evalScript sample.
--------------------------------------------------------------------------------

The evalScript sample demonstrates a one line embedding of Ejscript that can 
evaluate a script literal.

Files:
    main.c         - Main program that embeds Ejscript.

To build:
    make

To run:
    ./main

See Also:
    The evalFile and evalModule samples. See the native sample for how to create native classes.
